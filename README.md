# Safety micro class

安全微课自动刷课助手考试

# 使用方法

1、在main.py中输入tenantCode，userId，x_token，userProjectId四个参数值。

2、可以通过f12的网络栏获取（tenantCode，userId，x_token，userProjectId）。

由于安全微伴有调试检测，将显示模式切换为手机模式即可（这样就可以随便刷！！！）

3、本脚本也适用于其他网课。。。
